# VFX / 모션 그래픽 (VFX & Motion Graphics)

시각 효과(VFX)와 모션 그래픽은 실제 촬영만으로는 구현하기 어려운 시각적 요소를 디지털 기술로 만들어내는 분야입니다. 현실적인 합성부터 추상적인 그래픽 애니메이션까지 넓은 범위를 포괄합니다.

---

## 1. 그린 스크린 / 크로마 키 (Green Screen / Chroma Key)

- **정의**: 초록색(또는 파란색) 배경 앞에서 촬영한 후, 해당 색을 제거하고 다른 배경으로 합성하는 기법
- **사용 상황**: 실제로 가기 어려운 장소, 위험한 환경, 또는 가상의 세계를 배경으로 사용할 때
- **감정적 효과**: 세계관 확장, 판타지적 경험, 스케일감
- **예시 장면**: 뉴스 앵커 뒤의 날씨 지도, SF 영화의 우주 공간 장면

## 2. 모션 트래킹 (Motion Tracking)

- **정의**: 영상 속 특정 포인트의 움직임을 추적하여 CG 요소를 자연스럽게 부착하는 기법
- **사용 상황**: 움직이는 장면에 3D 오브젝트나 텍스트를 고정시킬 때
- **감정적 효과**: 현실과 가상의 매끄러운 통합으로 신뢰감 형성
- **예시 장면**: 빌딩 벽면에 가상의 간판을 부착하여 카메라가 움직여도 자연스럽게 유지

## 3. 로토스코핑 (Rotoscoping)

- **정의**: 영상에서 특정 대상의 윤곽을 프레임별로 따내어 분리하는 기법
- **사용 상황**: 그린 스크린 없이 촬영된 인물을 배경에서 분리하거나, 특정 영역에만 효과를 적용할 때
- **감정적 효과**: 정밀한 합성으로 시각적 완성도 향상
- **예시 장면**: 배우 뒤의 배경만 교체하면서 머리카락 한 올까지 자연스럽게 분리

## 4. 파티클 이펙트 (Particle Effects)

- **정의**: 수많은 작은 입자를 시뮬레이션하여 불, 연기, 비, 먼지, 폭발 등을 만드는 기법
- **사용 상황**: 자연 현상이나 마법 효과 등 입자 기반의 시각 효과가 필요할 때
- **감정적 효과**: 경외감, 위험, 아름다움, 마법적 분위기
- **예시 장면**: 마법사가 손에서 불꽃을 만들어내는 판타지 장면

## 5. 컴포지팅 (Compositing)

- **정의**: 여러 레이어의 영상, CG, 이미지를 합쳐 하나의 최종 프레임을 만드는 과정
- **사용 상황**: 배경, 인물, 효과, 조명 등 개별 요소를 통합하여 완성 화면을 만들 때
- **감정적 효과**: 현실과 구분할 수 없는 매끄러운 시각적 경험
- **예시 장면**: 스튜디오 촬영 인물 + CG 도시 배경 + 파티클 비 효과를 하나로 합성

## 6. 타이틀 시퀀스 / 타이포그래피 (Title Sequence / Typography)

- **정의**: 영상의 제목, 출연진, 크레딧 등을 동적으로 디자인하여 표현하는 모션 그래픽
- **사용 상황**: 영상의 시작이나 끝에 스타일리시한 정보 전달이 필요할 때
- **감정적 효과**: 첫인상 형성, 장르 암시, 기대감 조성
- **예시 장면**: 스릴러 영화 오프닝에서 글자가 떨리며 나타나는 불안한 타이틀 시퀀스

## 7. 로어 서드 / 자막 그래픽 (Lower Third / Caption Graphics)

- **정의**: 화면 하단 1/3 영역에 이름, 직함, 장소 등 정보를 표시하는 그래픽
- **사용 상황**: 다큐멘터리, 뉴스, 인터뷰 영상에서 인물이나 장소를 소개할 때
- **감정적 효과**: 전문적 느낌, 정보의 명확한 전달, 공신력
- **예시 장면**: 인터뷰 영상에서 "김OO, 영화감독" 텍스트가 하단에 자연스럽게 등장

## 8. 트랜지션 모션 (Transition Motion)

- **정의**: 장면 전환 시 사용하는 동적인 그래픽 효과 (와이프, 줌, 글리치 등)
- **사용 상황**: 장면 전환에 에너지와 스타일을 더하고 싶을 때
- **감정적 효과**: 역동성, 현대적 감각, 에너지
- **예시 장면**: 여행 브이로그에서 빠른 줌 트랜지션으로 장소가 바뀌는 전환

## 9. 스크린 교체 (Screen Replacement)

- **정의**: 영상 속 TV, 모니터, 스마트폰 화면의 내용을 후반 작업에서 교체하는 기법
- **사용 상황**: 촬영 시 빈 화면으로 두었던 기기에 특정 콘텐츠를 넣을 때
- **감정적 효과**: 현실감, 서사적 정보 전달
- **예시 장면**: 인물이 보는 스마트폰 화면에 메시지 알림이 뜨는 장면을 후반에 합성

## 10. 속도 변환 / 타임 리맵핑 (Speed Ramping / Time Remapping)

- **정의**: 영상의 재생 속도를 구간별로 다르게 조정하는 기법 (슬로모션 ↔ 빠른 동작)
- **사용 상황**: 액션의 임팩트를 강조하거나, 특정 순간을 극적으로 늘릴 때
- **감정적 효과**: 긴장, 아름다움, 초현실적 시간 경험
- **예시 장면**: 격투 장면에서 주먹이 맞는 순간만 슬로모션으로 전환 후 다시 정상 속도

## 11. 디지털 메이크업 / 에이징 (Digital Makeup / Aging)

- **정의**: CG 기술로 배우의 외모를 변화시키는 기법 (주름 추가, 젊어지게, 부상 등)
- **사용 상황**: 캐릭터의 나이 변화, 부상, 변신 등을 표현할 때
- **감정적 효과**: 시간의 흐름에 대한 감동, 충격, 변화의 체감
- **예시 장면**: 같은 배우가 20대에서 80대까지의 인생을 연기하며 디지털로 노화 표현

## 12. 2D 애니메이션 / 모션 인포그래픽 (2D Animation / Motion Infographic)

- **정의**: 그래픽 요소(아이콘, 도형, 텍스트)에 움직임을 부여하여 정보를 시각적으로 전달하는 기법
- **사용 상황**: 복잡한 데이터, 과정, 개념을 쉽고 흥미롭게 설명할 때
- **감정적 효과**: 이해의 용이성, 흥미, 전문적 느낌
- **예시 장면**: 교육 영상에서 통계 데이터가 막대그래프 애니메이션으로 표현
