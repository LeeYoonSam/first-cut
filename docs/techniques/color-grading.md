# 색보정 / 그레이딩 (Color Grading)

영상의 색상, 명암, 대비를 조정하여 특정 분위기와 감정을 만들어내는 후반 작업입니다. 색보정(Color Correction)은 기술적 정확성을, 색 그레이딩(Color Grading)은 예술적 표현을 목표로 합니다.

---

## 1. 색 온도 조정 (Color Temperature Adjustment)

- **정의**: 영상의 전체적인 색감을 따뜻하게(오렌지/노란색) 또는 차갑게(파란색) 조정하는 기법
- **사용 상황**: 장면의 시간대, 계절, 감정적 톤을 시각적으로 표현할 때
- **감정적 효과**: 따뜻한 톤은 친밀감/향수를, 차가운 톤은 고독/긴장을 유발
- **예시 장면**: 여름 회상 장면에 따뜻한 금빛 톤 적용, 겨울 밤 장면에 푸른 톤 적용

## 2. 틸/오렌지 룩 (Teal and Orange Look)

- **정의**: 그림자에 틸(청록)색을, 피부톤/하이라이트에 오렌지색을 적용하는 할리우드 표준 색감
- **사용 상황**: 시각적으로 세련되고 영화적인 느낌을 만들고 싶을 때
- **감정적 효과**: 전문적이고 세련된 영화적 느낌, 인물과 배경의 분리
- **예시 장면**: 도시 야경에서 네온 불빛(틸)과 인물의 피부(오렌지)가 대비되는 장면

## 3. 디새추레이션 / 채도 낮추기 (Desaturation)

- **정의**: 영상의 전체 또는 특정 색상의 채도(선명도)를 낮추는 기법
- **사용 상황**: 우울함, 과거 회상, 디스토피아 세계관을 표현할 때
- **감정적 효과**: 무기력함, 슬픔, 비현실감, 과거의 빛바랜 느낌
- **예시 장면**: 전쟁 영화에서 전투 장면의 채도를 낮춰 암울하고 건조한 분위기 연출

## 4. 하이 콘트라스트 (High Contrast)

- **정의**: 밝은 부분과 어두운 부분의 차이를 극대화하는 기법
- **사용 상황**: 드라마틱한 느낌, 필름 누아르 스타일, 또는 강렬한 감정을 표현할 때
- **감정적 효과**: 긴장, 드라마, 대립, 시각적 강렬함
- **예시 장면**: 범죄 드라마에서 심문실의 강한 조명과 짙은 그림자 대비

## 5. 로우 콘트라스트 (Low Contrast)

- **정의**: 밝은 부분과 어두운 부분의 차이를 줄여 부드러운 톤을 만드는 기법
- **사용 상황**: 몽환적, 평화로운, 또는 로맨틱한 분위기를 원할 때
- **감정적 효과**: 부드러움, 꿈결 같은 느낌, 평온, 향수
- **예시 장면**: 로맨스 영화의 첫 만남 장면에서 부드럽고 빛나는 색감 적용

## 6. 모노크롬 / 흑백 (Monochrome / Black and White)

- **정의**: 색상을 완전히 제거하고 명암만으로 영상을 표현하는 기법
- **사용 상황**: 예술적 표현, 과거 시대 재현, 또는 형태와 빛에 집중하고 싶을 때
- **감정적 효과**: 시대적 분위기, 예술성, 본질에 대한 집중, 감정의 정제
- **예시 장면**: 과거 회상 장면을 흑백으로 처리하여 현재와 명확히 구분

## 7. 선택적 색보정 (Selective Color Grading)

- **정의**: 영상의 특정 색상 범위만 선택하여 조정하는 기법 (예: 하늘의 파란색만 강화)
- **사용 상황**: 특정 요소를 시각적으로 강조하거나, 전체적 조화를 유지하면서 포인트를 줄 때
- **감정적 효과**: 시선 유도, 상징적 의미 부여, 시각적 세련됨
- **예시 장면**: 흑백에 가까운 화면에서 빨간 드레스만 색상을 유지하여 강조

## 8. LUT 적용 (Look-Up Table Application)

- **정의**: 미리 만들어진 색상 변환 테이블을 영상에 적용하여 일관된 색감을 만드는 기법
- **사용 상황**: 여러 장면이나 카메라로 촬영한 영상의 색감을 통일할 때, 또는 특정 스타일을 빠르게 적용할 때
- **감정적 효과**: 시각적 일관성, 전문적 품질감
- **예시 장면**: 시네마틱 LUT를 적용하여 일반 영상에 영화 같은 색감을 입힘

## 9. 비네팅 (Vignetting)

- **정의**: 프레임의 가장자리를 어둡게 하여 중앙의 피사체에 시선을 집중시키는 기법
- **사용 상황**: 인물에 시선을 집중시키거나, 빈티지/영화적 분위기를 더할 때
- **감정적 효과**: 집중, 친밀감, 복고적 느낌, 고전적 분위기
- **예시 장면**: 인물 클로즈업에 자연스러운 비네팅을 추가하여 시선을 얼굴에 집중

## 10. 스플릿 토닝 (Split Toning)

- **정의**: 영상의 어두운 부분(섀도)과 밝은 부분(하이라이트)에 서로 다른 색상을 적용하는 기법
- **사용 상황**: 영상에 독특한 색감과 분위기를 만들고 싶을 때
- **감정적 효과**: 독특한 스타일, 감정적 깊이, 시각적 풍부함
- **예시 장면**: 그림자에 깊은 푸른색, 하이라이트에 따뜻한 황금색을 적용한 일몰 장면

## 11. 블리치 바이패스 (Bleach Bypass)

- **정의**: 필름 현상 과정에서 표백(블리치) 단계를 건너뛴 것처럼 만드는 효과로, 채도가 낮고 콘트라스트가 높은 메탈릭한 질감
- **사용 상황**: 전쟁, 스릴러, 하드보일드 장르에서 거칠고 강렬한 시각적 톤을 원할 때
- **감정적 효과**: 거침, 날것의 느낌, 긴장, 위험
- **예시 장면**: 전쟁 영화에서 전투 장면에 블리치 바이패스를 적용하여 금속적이고 거친 질감 연출

## 12. 파워 윈도우 / 마스킹 (Power Window / Masking)

- **정의**: 영상의 특정 영역만 선택하여 독립적으로 색보정하는 기법
- **사용 상황**: 인물의 피부톤만 따로 조정하거나, 배경의 특정 부분만 밝기를 변경할 때
- **감정적 효과**: 자연스러운 시선 유도, 정밀한 분위기 조절
- **예시 장면**: 인물 주변만 밝게 하고 나머지를 어둡게 처리하여 스포트라이트 효과
