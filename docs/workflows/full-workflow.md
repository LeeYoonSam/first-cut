# 전체 워크플로우 상세 가이드

first-cut 프로젝트의 전체 영상 기획 워크플로우를 단계별로 설명합니다.

---

## 워크플로우 개요

```
사용자 입력 → /first-cut → 기획 단계 → 상세 설계 → 내보내기
                  │
                  ├── /pre-production (사전 기획)
                  ├── /benchmark (레퍼런스 분석)
                  ├── /scene-builder (장면 설계)
                  │       │
                  │       └── 기법 스킬 참조
                  │           ├── 촬영 기법
                  │           ├── 연출 기법
                  │           ├── 편집 기법
                  │           ├── 사운드 디자인
                  │           ├── 색보정
                  │           ├── VFX/모션
                  │           ├── 사전 기획
                  │           └── 장르별 기법
                  │
                  ├── /export-plan (기획서 내보내기)
                  └── /notion-sync (노션 연동)
```

---

## 1단계: 프로젝트 시작 (/first-cut)

영상 기획의 시작점입니다. AI가 대화를 통해 영상의 기본 방향을 파악합니다.

**수집하는 정보:**
- 영상 유형 (브이로그, 광고, 뮤직비디오, 다큐멘터리 등)
- 영상 목적 (홍보, 교육, 엔터테인먼트 등)
- 타겟 관객
- 희망 길이
- 원하는 분위기/톤

**예시 대화:**

```
사용자: /first-cut

AI: 안녕하세요! 영상 기획을 시작하겠습니다.
    어떤 종류의 영상을 만들고 싶으신가요?

사용자: 제품 소개 영상을 만들려고 해요. 새로 나온 무선 이어폰이에요.

AI: 좋습니다! 몇 가지 더 여쭤볼게요.
    1. 이 영상은 어디에 게시할 예정인가요? (유튜브, 인스타그램, 등)
    2. 영상 길이는 어느 정도를 생각하시나요?
    3. 참고하고 싶은 영상이나 브랜드가 있나요?

사용자: 유튜브랑 인스타 릴스 둘 다요. 30초~1분 정도. 애플 광고 느낌이면 좋겠어요.
```

---

## 2단계: 사전 기획 (/pre-production)

1단계에서 수집한 정보를 바탕으로 상세 기획을 진행합니다.

**생성되는 내용:**
- 영상 컨셉 문서
- 무드보드 키워드
- 촬영 장소 제안
- 필요 장비 목록
- 대략적인 촬영 일정

**예시 대화:**

```
사용자: /pre-production

AI: 앞서 기획한 "무선 이어폰 제품 소개 영상"을 기반으로
    사전 기획을 진행하겠습니다.

    [컨셉]
    미니멀하고 세련된 느낌의 제품 영상
    "일상 속 완벽한 사운드" 테마

    [추천 촬영 장소]
    - 깔끔한 화이트/그레이 배경 스튜디오
    - 도시 카페 (일상 사용 장면)
    - 러닝/운동 공간 (활동적 사용 장면)

    [필요 장비]
    - 카메라: 4K 촬영 가능 미러리스
    - 렌즈: 매크로 렌즈 (제품 클로즈업용)
    - 조명: 소프트박스 2개, LED 패널 1개
    - ...
```

---

## 3단계: 레퍼런스 분석 (/benchmark)

참고할 영상이나 스타일을 분석합니다.

**분석 항목:**
- 촬영 기법 분석 (어떤 앵글, 움직임을 사용했는지)
- 편집 패턴 (컷 리듬, 전환 효과)
- 색감/그레이딩 스타일
- 사운드 구성
- 전체적인 템포와 구조

**예시 대화:**

```
사용자: /benchmark
        애플 에어팟 프로 광고 스타일을 분석해줘

AI: 애플 에어팟 프로 광고 스타일 분석:

    [촬영 기법]
    - 극단적 클로즈업: 제품 디테일 강조
    - 매크로 촬영: 소재 질감 표현
    - 슬로모션: 제품 착용 순간 강조
    - 와이드 샷: 사용 환경 제시

    [편집]
    - 빠른 컷 편집 (0.5~2초 단위)
    - 비트 싱크 편집 (음악에 맞춘 전환)
    - 매치 컷 (형태 유사성 활용)

    [색감]
    - 깨끗하고 밝은 톤
    - 약간의 디새추레이션
    - 하이라이트 강조
    ...
```

---

## 4단계: 장면 설계 (/scene-builder)

개별 장면(씬)을 구체적으로 설계합니다. 이 단계에서 기법 레퍼런스 문서를 참조합니다.

**각 장면에 포함되는 정보:**
- 장면 번호와 설명
- 촬영 기법 (앵글, 카메라 움직임)
- 조명 설정
- 사운드 설계
- 편집 방향
- 예상 길이

**예시 대화:**

```
사용자: /scene-builder
        오프닝 장면을 설계해줘

AI: [씬 #1: 오프닝 - "일상의 소음"]
    길이: 5초
    설명: 도시의 소음이 가득한 환경에서 시작

    촬영 기법:
    - 와이드 샷으로 번화한 거리 촬영
    - 핸드헬드로 약간의 흔들림 (현실감)
    - 하이 앵글에서 로우 앵글로 전환

    사운드:
    - 도시 앰비언스 (교통 소음, 사람 소리)
    - 볼륨 점점 증가

    편집:
    - 점프 컷으로 다양한 소음 원 빠르게 전환
    - 마지막에 스매시 컷으로 정적 전환
    ...
```

---

## 5단계: 기획서 내보내기 (/export-plan)

완성된 기획을 정리된 마크다운 문서로 내보냅니다.

**내보내기 형식:**
- 전체 기획 요약서
- 장면별 상세 기획 (씬 리스트)
- 촬영 체크리스트
- 장비 목록
- 일정 제안

---

## 6단계: 노션 연동 (/notion-sync) (선택)

기획서를 노션으로 동기화하여 팀과 공유할 수 있습니다. 설정 방법은 [노션 연동 가이드](notion-setup.md)를 참고하세요.
