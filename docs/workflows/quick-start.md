# 빠른 시작 가이드 - 5분 안에 첫 기획서 만들기

이 가이드를 따라하면 5분 안에 영상 기획서를 만들 수 있습니다.

---

## 사전 준비

1. **Claude Code** 설치가 완료되어 있어야 합니다
2. 터미널(명령줄)을 열 수 있어야 합니다

---

## 단계별 진행

### 1단계: 프로젝트 클론 (1분)

```bash
git clone <프로젝트-저장소-URL>
cd first-cut
```

### 2단계: Claude Code 실행 (30초)

```bash
claude
```

Claude Code가 실행되면 프로젝트의 스킬들이 자동으로 로드됩니다.

### 3단계: /first-cut 실행 (30초)

Claude Code 프롬프트에서 다음을 입력합니다:

```
/first-cut
```

이 명령어가 영상 기획 워크플로우를 시작합니다.

### 4단계: 질문에 응답 (2분)

AI가 영상에 대한 질문을 합니다. 간단히 답변해 주세요:

**예시 대화:**

```
AI: 어떤 종류의 영상을 만들고 싶으신가요?
사용자: 카페 브이로그 영상이요.

AI: 영상의 목적은 무엇인가요?
사용자: 새로 오픈한 카페를 소개하려고요.

AI: 영상 길이는 어느 정도를 생각하시나요?
사용자: 3분 정도요.

AI: 특별히 원하는 분위기나 스타일이 있나요?
사용자: 따뜻하고 아늑한 느낌이요.
```

### 5단계: 기획서 확인 (1분)

AI가 자동으로 기획서를 생성합니다. 기획서에는 다음이 포함됩니다:

- 영상 컨셉 요약
- 추천 촬영 기법 (예: 클로즈업, 슬로모션)
- 장면별 구성 (씬 리스트)
- 추천 색감/분위기
- 사운드 디자인 제안
- 편집 스타일 가이드

---

## 다음 단계

기획서가 마음에 들면 더 상세한 작업을 진행할 수 있습니다:

- `/pre-production` - 사전 기획을 더 상세하게 발전시키기
- `/scene-builder` - 개별 장면을 구체적으로 설계하기
- `/benchmark` - 레퍼런스 영상 분석하기
- `/export-plan` - 최종 기획서를 마크다운으로 내보내기

전체 워크플로우에 대한 상세 설명은 [전체 워크플로우 가이드](full-workflow.md)를 참고하세요.

---

## 문제 해결

| 문제 | 해결 방법 |
|------|-----------|
| Claude Code가 실행되지 않음 | `npm install -g @anthropic-ai/claude-code`로 재설치 |
| /first-cut 명령이 인식되지 않음 | 프로젝트 디렉토리(first-cut/) 안에서 실행했는지 확인 |
| 스킬이 로드되지 않음 | `.claude/skills/` 디렉토리에 스킬 파일이 있는지 확인 |
