# 새 기법 추가 절차

first-cut 프로젝트에 새로운 영상 제작 기법을 추가하는 방법을 단계별로 설명합니다.

---

## 수정 파일 목록

새 기법을 추가하려면 다음 파일들을 수정해야 합니다:

1. **스킬 파일**: `.claude/skills/{카테고리}.md`
2. **기법 레퍼런스**: `docs/techniques/{카테고리}.md`
3. **용어집**: `docs/glossary.md`
4. **README**: `README.md` (필요 시)

---

## 단계별 체크리스트

### 1단계: 기법 카테고리 결정

새 기법이 어떤 카테고리에 속하는지 결정합니다:

- [ ] 사전 기획 (pre-production)
- [ ] 촬영 기법 (cinematography)
- [ ] 연출 기법 (directing)
- [ ] 편집 기법 (editing)
- [ ] 사운드 디자인 (sound-design)
- [ ] 색보정/그레이딩 (color-grading)
- [ ] VFX/모션 그래픽 (vfx-motion)
- [ ] 장르별 기법 (genre-specific)

### 2단계: 스킬 파일 수정

`.claude/skills/{카테고리}.md` 파일에 새 기법을 추가합니다.

**추가할 내용:**
- 기법명 (영문 + 한국어)
- 기법 설명
- 적용 조건/상황
- 관련 기법 문서 참조 링크

### 3단계: 기법 레퍼런스 문서 수정

`docs/techniques/{카테고리}.md` 파일에 새 기법 항목을 추가합니다.

**필수 포함 내용:**

```markdown
## N. 기법명 (English Name)

- **정의**: 기법에 대한 명확한 설명
- **사용 상황**: 어떤 상황에서 이 기법을 사용하는지
- **감정적 효과**: 관객에게 주는 감정적/심리적 효과
- **예시 장면**: 실제 영상에서의 구체적인 사용 예시
```

**작성 기준:**
- 영상 초보자도 이해할 수 있는 쉬운 언어로 작성
- 기법명은 영문과 한국어를 함께 표기
- 예시 장면은 구체적이고 상상하기 쉬운 것으로 선택

### 4단계: 용어집 업데이트

`docs/glossary.md`에 새 기법의 용어를 추가합니다.

**추가 형식:**

```markdown
### 기법명 (English Name)
→ [관련 문서](techniques/{카테고리}.md) 참고. 간단한 설명 한 줄.
```

**배치 규칙:**
- 한글 용어는 가나다 순으로 배치
- 영문 용어는 A-Z 순으로 배치
- 관련 기법 문서로의 링크 포함

### 5단계: 링크 검증

- [ ] 스킬 파일에서 기법 문서로의 링크가 올바른지 확인
  - 경로: `../../docs/techniques/{카테고리}.md`
- [ ] 기법 문서에서 스킬 파일로의 링크가 올바른지 확인
  - 경로: `../../.claude/skills/{스킬명}.md`
- [ ] 용어집에서 기법 문서로의 링크가 올바른지 확인
  - 경로: `techniques/{카테고리}.md`

### 6단계: 최종 점검

- [ ] 기법 설명이 초보자도 이해할 수 있는 수준인가?
- [ ] 기법명에 영문과 한국어가 모두 포함되어 있는가?
- [ ] 예시 장면이 구체적이고 이해하기 쉬운가?
- [ ] 모든 링크가 올바르게 작동하는가?
- [ ] 용어집의 정렬 순서가 맞는가?

---

## 예시: "더블 익스포저" 기법 추가

### 1. 카테고리: VFX/모션 그래픽

### 2. 스킬 파일 수정 (`.claude/skills/vfx-motion-graphics.md`)

```markdown
### 더블 익스포저 (Double Exposure)
두 개의 영상을 겹쳐 합성하는 기법.
인물의 실루엣 안에 풍경을 넣는 등 몽환적 표현에 활용.
참고: [VFX 기법 문서](../../docs/techniques/vfx-motion.md)
```

### 3. 기법 문서 수정 (`docs/techniques/vfx-motion.md`)

```markdown
## 13. 더블 익스포저 (Double Exposure)

- **정의**: 두 개의 영상이나 이미지를 겹쳐서 하나의 프레임에 합성하는 기법
- **사용 상황**: 인물의 내면 세계, 기억, 감정을 시각적으로 표현할 때
- **감정적 효과**: 몽환, 시적 아름다움, 내면과 외면의 이중성
- **예시 장면**: 인물의 실루엣 안에 숲의 풍경이 겹쳐 보이는 뮤직비디오 장면
```

### 4. 용어집 수정 (`docs/glossary.md`)

```markdown
### 더블 익스포저 (Double Exposure)
→ [VFX 문서](techniques/vfx-motion.md) 참고. 두 영상을 겹쳐 합성하는 기법.
```
