## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 영상 제목 | "같은 물, 다른 수모" — 커플 수모 홍보 영상 |
| 장르 | 제품 소개 광고 |
| 타겟 오디언스 | 전 연령, 귀여운 디자인을 선호하는 수영인, 커플 |

- 자체 디자인 커플 수모 (수영모자)
- 수영하는 남자 / 수영하는 여자를 **실선(라인 드로잉)**으로 그린 디자인
- 심플하면서 한눈에 커플임을 알 수 있음
- 아트적인 감성의 라인 드로잉

## 2. 콘셉트 & 무드

**핵심 키워드**: 실선 드로잉, 커플, 유머, 감각적, 수영장

**무드보드 설명**: 밝고 에너지 넘치는 수영장 배경 위에 심플한 라인 아트 디자인이 돋보이는 영상. 유머러스한 상황극 속에서 제품의 디자인적 매력을 자연스럽게 보여주되, 과하지 않은 세련된 웃음을 추구한다. 색감은 전반적으로 밝고 선명하며, 제품 공개 전후로 색감 대비를 통해 반전 효과를 극대화한다.

**레퍼런스 영상**: 별도 레퍼런스 없음. 아래 벤치마크 분석 참고.

## 4. 씬별 상세 기획

### 씬 1: 후킹 (Hook) — "커플인 거 어떻게 티 내지?"

- **설명**: 수영장 입구 앞에서 커플이 고민하는 상황. "수영장에서 커플인 거... 어떻게 티 내지?"
- **목적**: 시청자의 관심을 잡는 후킹 (첫 3초 승부)
- **예상 길이**: ~8초

#### 촬영 기법
- 샷 사이즈(Shot Size): MS(미디엄샷) — 두 사람의 표정과 관계를 동시에 보여줌
- 앵글(Angle): 아이레벨(Eye Level, 눈높이 촬영) — 시청자와 동등한 시선으로 공감 유도
- 카메라 무브먼트: 고정샷 또는 가벼운 핸드헬드(Handheld, 손으로 들고 찍기) — 일상적이고 자연스러운 느낌
- 구도: 3분할 구도(Rule of Thirds) — 두 사람을 좌우에 배치
- 조명: 자연광(실외) — 밝고 현실적

#### 사운드
- BGM: 경쾌한 팝/재즈 — 밝은 피아노, 관악기, 코믹한 느낌 (볼륨 낮게 시작)
- SFX(효과음): 고민 순간 "띵~" 또는 "뚜둥" 코믹 효과음
- 앰비언스(배경음): 수영장 주변 소리 (물소리, 사람들 웅성거림) 낮게

#### 편집 기법
- 페이싱: 빠르게 — 3~4초 안에 상황 설정, 나머지로 고민 표정
- 컷 방식: 스트레이트 컷(Straight Cut, 효과 없이 바로 연결하는 기본 컷)
- 다음 씬 전환: 점프컷(Jump Cut, 같은 공간에서 시간을 생략하며 연결) → 씬 2로

#### 색보정
- 컬러 톤: 밝고 선명한 자연광 톤 — 채도 중간~높음, 대비 중간
- 색온도: 자연광 계열 (5500~6500K) — 여름 수영장 분위기

#### 연출 노트
- 첫 대사가 후킹의 핵심. 자연스럽고 공감 가는 톤으로 연기
- 수영장 입구가 배경에 살짝 보이면 장소 설정에 도움

### 씬 2: 문제 제기 (Problem) — "다 똑같잖아..."

- **설명**: 커플이 평범한 검정 수모를 쓰고 수영장에 있지만, 주변 사람들과 구분이 안 됨. 잘못된 사람에게 손 흔드는 실수
- **목적**: 평범한 수모의 답답한 문제 상황을 유머러스하게 보여줌
- **예상 길이**: ~12초

#### 촬영 기법
- 샷 사이즈: LS(롱샷, 멀리서 넓게 찍기) → MS(미디엄샷) — 비슷비슷한 수모 전경 후 당황하는 커플로 전환
- 앵글: 아이레벨 — 관객도 함께 "누가 누군지 모르겠다"를 체험
- 카메라 무브먼트: 가벼운 팬(Pan, 카메라를 좌우로 돌리며 훑기) — 비슷한 수모들을 쭉 훑기
- 구도: 3분할 구도 — 사람들 속에 묻힌 커플 강조
- 조명: 실내 수영장 조명 또는 실외 자연광

#### 사운드
- BGM: 씬 1에서 이어지는 경쾌한 BGM, 살짝 템포 다운 (답답함 연출)
- SFX: 두리번거릴 때 "삐빅~(오답)" / 잘못 손 흔들 때 "찌이익~(브레이크)" 효과음
- 앰비언스: 수영장 실내 반향음, 물 첨벙 소리

#### 편집 기법
- 페이싱: 중간 — 2~3초 컷으로 상황 보여주기
- 컷 방식: 빠른 스트레이트 컷 — LS 전경 → 두리번 CU → 잘못 손 흔드는 MS → 민망 리액션 CU
- 포인트: 잘못 손 흔드는 장면에서 0.5초 프리즈 프레임(Freeze Frame, 영상 일시 정지) + 자막 "다 똑같잖아..."
- 다음 씬 전환: 스매시컷(Smash Cut, 갑작스럽게 확 바뀌는 전환) → 씬 3 수모 클로즈업으로

#### 색보정
- 컬러 톤: 채도 살짝 낮춤 — 밋밋하고 평범한 세계를 시각적으로 표현
- 색온도: 약간 쿨톤 (4500~5000K) — 씬 3의 반전과 대비 효과

#### 연출 노트
- 주변 엑스트라들의 수모는 모두 비슷한 검정/남색 계열로 통일
- 잘못 손 흔드는 민망한 순간이 유머의 핵심 — 타이밍과 표정 연기가 중요
- 프리즈 프레임 + 자막 조합으로 유머 극대화

### 씬 3: 반전 등장 (Solution) — 실선 드로잉 수모 공개

- **설명**: 수모 두 개가 나란히 놓여 있다. 실선으로 그려진 수영하는 남자/여자 디자인이 클로즈업으로 공개. BGM 전환과 함께 분위기 반전
- **목적**: 제품의 아트적 디자인을 감각적으로 공개
- **예상 길이**: ~12초

#### 촬영 기법
- 샷 사이즈: ECU(익스트림 클로즈업, 극도로 가까이 찍기) → CU(클로즈업) — 실선 드로잉 디테일 먼저, 점점 전체 모습 공개
- 앵글: 하이앵글(High Angle, 위에서 내려다보기) — 테이블 위 수모를 감각적으로 촬영, 제품 사진 느낌
- 카메라 무브먼트: 느린 푸시인(Dolly In, 카메라가 천천히 다가가기) — 디자인에 시선 집중
- 구도: 중앙 구도(Center Composition) — 수모 두 개를 화면 중앙에 대칭 배치, 한 쌍 강조
- 조명: 하이키 조명(High Key, 밝고 균일한 조명) — 깨끗하고 선명하게 디자인 보여줌

#### 사운드
- BGM: 스매시컷과 동시에 전환 — 세련된 일렉트로닉 팝 또는 감각적 비트로 분위기 확 바꾸기
- SFX: 수모 등장 순간 "쨘!" 임팩트 사운드(whoosh) / 집어 들 때 가벼운 천 소리
- 앰비언스: 최소화 — 제품에 집중하도록 주변 소리 줄이기

#### 편집 기법
- 페이싱: 느리게 → 빠르게 — 처음 5초 디테일 천천히, 후반 7초 템포 UP
- 컷 방식: ECU 디테일 → CU 전체 → MS 커플이 집어 드는 컷으로 확장
- 포인트: 남자/여자 수모를 스플릿 스크린(Split Screen, 화면 분할)으로 동시 비교도 효과적
- 다음 씬 전환: 매치컷(Match Cut, 형태가 비슷한 것을 연결) — 손에 든 수모 → 머리에 쓴 수모

#### 색보정
- 컬러 톤: 씬 2 대비 채도 확 올리기 — 밝고 선명, "세상이 달라졌다" 느낌
- 색온도: 약간 웜톤 (5500K+) — 따뜻하고 감각적
- 포인트: **씬 2(쿨톤, 낮은 채도) → 씬 3(웜톤, 높은 채도)**의 색감 대비가 반전의 핵심

#### 연출 노트
- 이 씬은 영상의 전환점. 스매시컷 + BGM 전환 + 색감 변화를 동시에 사용해 최대 임팩트
- 실선 드로잉의 디테일이 잘 보이도록 조명과 초점에 특히 신경 쓸 것
- 중앙 대칭 구도로 "한 쌍"의 느낌을 시각적으로 강조

### 씬 4: 하이라이트 (Fun) — 커플 수모의 위력

- **설명**: 커플이 수모를 쓰고 수영장에 등장. 슬로우모션 런웨이, 주변 부러움 리액션, 커플 장난 장면들
- **목적**: 제품 착용 후 유머 상황들로 즐거움과 부러움 유발
- **예상 길이**: ~18초

#### 세부 장면 구성

1. **등장 순간** (~5초): 커플이 자신감 넘치게 걸어 나옴. 슬로우모션 런웨이 과장. 주변 수영인들 쳐다봄
2. **주변 리액션** (~5초): 다른 수영인이 민무늬 수모 벗어서 한숨 / 친구끼리 "우리도 저거 사자" 표정 / 솔로 수영인 부러운 눈
3. **커플 장난** (~8초): 나란히 수영 후 서로의 수모 가리키며 웃음 / 물속에서 올라와 수모 맞대는 포즈 (남자+여자 드로잉이 마주보는 구도) / 엄지척

#### 촬영 기법
- 샷 사이즈: MLS(미디엄롱샷, 전신이 거의 보이는 샷) → MS → CU 혼합 — 다양한 사이즈로 역동적
- 앵글: 아이레벨 기본 + 등장 순간 로우앵글(Low Angle, 아래에서 위로 올려다보기) — 과장된 등장감
- 카메라 무브먼트: 핸드헬드 — 현장감, 에너지 / 등장 장면은 슬로우모션 트래킹
- 구도: 3분할 + 중앙 — 커플 중앙, 리액션 인물 양쪽
- 조명: 자연광 또는 수영장 조명 — 밝고 에너지 넘치게

#### 사운드
- BGM: 업비트 팝 — 영상에서 가장 에너지 높은 구간. 비트에 맞춰 컷 전환
- SFX: 등장 "휘이~" 바람 소리 / 수모 벗기 "슈웅~" 허탈 / 엄지척 "따단!" 짧은 팡파르
- 앰비언스: 수영장 물소리 + 사람 소리 (활기찬 분위기)

#### 편집 기법
- 페이싱: 빠르게 — 1~3초 컷, 음악 비트에 맞춰 컷 전환 (비트 싱크)
- 컷 방식: 빠른 스트레이트 컷 연속 + 등장 순간 슬로우모션 (0.5배속)
- 포인트: 수모 맞대는 포즈에서 프리즈 프레임 — 남자/여자 드로잉이 마주보는 순간 강조
- 다음 씬 전환: BGM 볼륨 다운 + 디졸브(Dissolve, 서서히 겹치며 전환) → 씬 5로

#### 색보정
- 컬러 톤: 영상에서 가장 밝고 선명 — 채도 높음, 대비 높음
- 색온도: 웜톤 유지 (5500K+) — 에너지와 활력
- 포인트: 슬로우모션 구간만 골든톤(Golden Tone, 황금빛) 추가 — 특별한 순간 강조

#### 연출 노트
- 슬로우모션 등장은 일부러 과장해서 코믹하게 연출 (진지한 표정 + 런웨이 워킹)
- 주변 리액션 연기가 자연스러워야 유머가 살아남 — 과하지 않게
- 수모를 맞대는 포즈는 남자/여자 드로잉이 서로 마주보도록 위치 조정 필요

### 씬 5: 마무리 (CTA) — 감성 클로징 + 구매 유도

- **설명**: 커플이 수영장 가장자리에 나란히 앉아 있는 감성 장면 → 제품 컷 → 물에 뛰어드는 엔딩
- **목적**: 감성적 마무리 + 브랜드/구매 정보 전달
- **예상 길이**: ~10초

#### 촬영 기법
- 샷 사이즈: MS(나란히 앉은 커플) → CU(수모 디테일 제품 컷) → LS(물에 뛰어드는 뒷모습)
- 앵글: 백샷(뒤에서 촬영) — 커플의 나란한 실루엣과 수모가 함께 보이도록 / 제품 컷은 하이앵글
- 카메라 무브먼트: 고정샷 — 차분하고 안정적, 여운
- 구도: 중앙 구도 — 커플 대칭 + 제품 대칭, 마무리의 안정감
- 조명: 골든아워 자연광(Golden Hour, 일몰 전후 따뜻한 황금빛) — 감성적 마무리 / 제품 컷은 하이키 조명

#### 사운드
- BGM: 템포 다운, 잔잔한 어쿠스틱 또는 피아노 아웃트로 — 여운 있게 마무리
- SFX: 물에 뛰어드는 "첨벙!" 유쾌한 엔딩 사운드 / 로고 등장 시 짧은 브랜드 사운드
- 앰비언스: 수영장 물소리 잔잔하게 — 평화로운 마무리

#### 편집 기법
- 페이싱: 느리게 — 5~8초 컷, 여운을 남기는 호흡
- 컷 방식: 디졸브로 부드럽게 연결 — 나란히 앉은 커플 → 제품 컷 → 뛰어드는 뒷모습
- 포인트: "첨벙" 직후 페이드 투 블랙(Fade to Black, 화면이 서서히 어두워지며 끝) + 로고/브랜드 정보

#### 색보정
- 컬러 톤: 따뜻한 골든톤 — 감성적이면서 기분 좋은 마무리
- 색온도: 고색온도 (5500K+, 오렌지·앰버 계열) — 골든아워 느낌
- 포인트: 제품 컷만 깨끗한 뉴트럴톤(Neutral Tone) — 디자인이 정확히 보이도록

#### 카피/자막
- 메인 카피: **"같은 물, 다른 수모"**
- 서브 카피: "나란히 달라지는 순간" 또는 "한 쌍이니까"
- CTA: 브랜드명 + 구매 링크 또는 QR코드 + SNS 계정

#### 연출 노트
- 골든아워 촬영이 어려우면 후보정으로 골든톤 추가 가능
- 마지막 "첨벙"은 유머러스한 영상의 마침표 — 밝고 유쾌한 에너지로 끝내기
- 브랜드 정보는 깔끔하게, 3초 이상 노출

```

  ▲
5 │                          ★ 씬4: 웃음/즐거움
  │                        ╱    ╲

  │                 놀라움╱        ╲
3 │                    ╱            ╲ ★ 씬5: 여운/구매욕
  │  ★ 씬1          ╱

  │        ╲ ★ 씬2╱

  │
  └──────────────────────────────────────▶ 시간
   0s    8s     20s     32s      50s   60s
```

- **0:00~0:08** 씬 1 (후킹): 공감과 호기심 — "나도 그런 고민 해봤는데"
- **0:08~0:20** 씬 2 (문제): 답답함과 민망함 — 유머로 풀어낸 문제 상황
- **0:20~0:32** 씬 3 (반전): 놀라움과 설렘 — 디자인 공개의 임팩트
- **0:32~0:50** 씬 4 (하이라이트): 웃음과 즐거움의 최고조 — 바이럴 포인트
- **0:50~1:00** 씬 5 (마무리): 따뜻한 여운 + 구매 욕구

## 7. 색보정/그레이딩 전체 가이드

### 전체 컬러 방향
- 밝고 에너지 넘치는 톤 기반, 씬별로 전략적 색감 변화
- 전반적으로 채도 중간~높음, 자연광 기반

### 씬별 색감 변화 전략

| 씬 | 색온도 | 채도 | 대비 | 의도 |
|----|--------|------|------|------|
| 씬 1 | 자연광 (5500~6500K) | 중간~높음 | 중간 | 밝은 일상 |
| 씬 2 | 쿨톤 (4500~5000K) | **낮음** | 중간 | 밋밋하고 답답한 세계 |
| 씬 3 | 웜톤 (5500K+) | **높음** | 중간~높음 | "세상이 달라졌다" 반전 |
| 씬 4 | 웜톤 (5500K+) | 높음 | 높음 | 최고의 에너지와 활력 |
| 씬 5 | 골든톤 (5500K+) | 중간 | 중간 | 따뜻한 감성 마무리 |

### 핵심 포인트
- **씬 2→3 색감 반전**이 영상의 가장 중요한 시각적 전환점
- 슬로우모션 구간은 골든톤 추가로 특별한 순간 강조
- 제품 컷(씬 3, 씬 5)은 뉴트럴톤으로 디자인이 정확히 보이도록 처리
- 피부톤은 항상 자연스럽게 보호 (스킨톤 범위: H:20-40, S:20-60)

### LUT 추천
- 모던 클린룩 (Technical LUT) 베이스 → 씬별 커스텀 보정 추가
- 스마트폰 촬영 시: VSCO, Lightroom Mobile 등 모바일 앱 활용 가능

- [ ] 수영장 촬영 허가 확인
