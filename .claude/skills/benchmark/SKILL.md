## 목적

주어진 영상 주제나 장르에 대해 인기 콘텐츠를 조사하고, 트렌드를 파악하여 기획에 활용할 수 있는 인사이트를 제공합니다.

## 사용 시점

- 사용자가 `/benchmark`를 호출했을 때
- 영상 기획 초기 단계에서 레퍼런스 콘텐츠 조사가 필요할 때
- 특정 장르나 플랫폼의 최신 트렌드를 파악해야 할 때
- 경쟁 채널 분석이 필요할 때

---

## 리서치 지침

### 1단계: 조사 범위 확인

사용자에게 다음을 확인하세요.
선택지가 있는 질문은 `AskUserQuestion`을 사용하여 클릭 가능한 옵션으로 제공하세요.

- **주제/키워드** (일반 텍스트): "어떤 주제로 조사할까요? (예: '요리 영상', '여행 브이로그')"

- **플랫폼** (AskUserQuestion 사용):
```
AskUserQuestion:
  question: "어떤 플랫폼을 기준으로 조사할까요?"
  header: "플랫폼"
  multiSelect: true
  options:
    - label: "유튜브"
      description: "YouTube 인기 영상 및 채널 분석"
    - label: "인스타그램 릴스"
      description: "Instagram Reels 트렌드 분석"
    - label: "틱톡"
      description: "TikTok 인기 콘텐츠 분석"
    - label: "기타"
      description: "네이버, 비메오 등 다른 플랫폼"
```

- **조사 범위** (AskUserQuestion 사용):
```
AskUserQuestion:
  question: "국내와 해외 콘텐츠 중 어디를 조사할까요?"
  header: "조사 범위"
  options:
    - label: "국내 중심"
      description: "한국어 콘텐츠와 국내 크리에이터 위주로 조사"
    - label: "해외 중심"
      description: "영어권 등 해외 콘텐츠 위주로 조사"
    - label: "국내 + 해외 모두"
      description: "국내외 콘텐츠를 폭넓게 조사"
```

- **특정 채널** (일반 텍스트): "참고할 특정 채널이나 크리에이터가 있나요? (없으면 건너뛰어도 됩니다)"

### 2단계: 웹 검색 수행

WebSearch 도구를 사용하여 다음을 조사하세요:

**인기 콘텐츠 조사**:
- `"{주제} 유튜브 인기 영상"` 검색
- `"{주제} YouTube trending"` 검색
- 조회수 높은 영상 3~5개 식별

**트렌드 조사**:
- `"{주제} 2025 트렌드"` 또는 `"{주제} trend 2025"` 검색
- 최근 6개월 내 급상승 포맷/스타일 파악

**경쟁 채널 분석**:
- 해당 분야 주요 채널 3~5개 파악
- 각 채널의 구독자 수, 평균 조회수, 업로드 빈도 확인

### 3단계: 각 레퍼런스 콘텐츠 분석

발견한 콘텐츠마다 다음 항목을 분석하세요:

| 분석 항목 | 내용 |
|-----------|------|
| 영상 제목/채널 | 이름과 링크 |
| 전체 구조 | 인트로→본문→아웃트로 구성 방식 |
| 사용된 기법 | 주요 촬영·편집·사운드 기법 |
| 차별화 포인트 | 다른 영상과 구별되는 독특한 요소 |
| 반응/성과 | 조회수, 좋아요 수, 댓글 분위기 |
| 영상 길이 | 총 길이와 씬별 배분 |

### 4단계: 인사이트 도출

분석을 바탕으로 다음을 정리하세요:

- **공통 성공 패턴**: 인기 영상들이 공유하는 구조나 기법
- **차별화 기회**: 아직 잘 활용되지 않은 접근 방식
- **주의할 점**: 이미 과포화된 방식, 피해야 할 클리셰
- **트렌드 반영 제안**: 현재 유행하는 요소 중 프로젝트에 적용 가능한 것

## 결과 저장

분석 결과를 다음 형식으로 `plans/drafts/{프로젝트명}-benchmark.md`에 저장하세요:

```markdown
# 벤치마크 분석: {주제/장르}
분석 일자: YYYY-MM-DD

## 레퍼런스 콘텐츠

### 1. {채널명} - {영상 제목}
- 구조: ...
- 주요 기법: ...
- 차별화 포인트: ...
- 성과: 조회수 X만, 좋아요 X천

### 2. ...

## 트렌드 요약

## 인사이트 및 적용 제안
```

---

## 용어 설명

- **벤치마크(Benchmark)**: 기준점이 되는 참고 콘텐츠. 우리 영상의 목표 수준을 가늠하는 잣대
- **클리셰(Cliché)**: 너무 많이 써서 진부해진 표현이나 기법
- **인사이트(Insight)**: 데이터 분석을 통해 얻은 핵심 깨달음이나 아이디어
