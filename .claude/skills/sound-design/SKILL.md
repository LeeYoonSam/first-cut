## 목적

장면의 감정적 목표와 분위기를 분석하여 BGM 장르, SFX(효과음), 나레이션 방식, 앰비언스를 결정한다. 음악 저작권 및 라이선스 주의사항도 포함한다.

## 사용 시점

- 영상 편집 중 음악/사운드를 선택할 때
- 장면 분위기에 맞는 BGM 장르를 결정할 때
- SFX 레이어링 구성을 계획할 때
- 나레이션 톤과 스타일을 결정할 때
- 오디오 믹싱 우선순위를 정할 때

## 결정 매트릭스

### BGM 장르 선택

| 장면 분위기/목적 | 추천 BGM 장르 | 악기 구성 | 예시 |
|----------------|--------------|-----------|------|
| 감성 여행·힐링·일상 | 어쿠스틱 팝 / 인디 포크 | 통기타, 피아노, 가벼운 퍼커션 | 여행 브이로그, 일상 기록 |
| 동기부여·성취·스포츠 | 업비트 팝 / 일렉트로닉 | 신디사이저, 드럼, 베이스 | 운동 영상, 성공 스토리 |
| 긴장·스릴·추격 | 오케스트라 스트링 / 다크 앰비언트 | 현악기, 저음 타악기, 금관 | 스릴러, 액션 |
| 슬픔·이별·상실 | 미니멀 피아노 / 클래식 | 피아노 솔로, 첼로 | 이별 장면, 추모 |
| 공포·불안·미스터리 | 다크 앰비언트 / 아토날 | 불규칙 사운드, 저주파, 드론 | 공포 영상, 미스터리 |
| 웅장·서사·역사 | 오케스트라 풀 스코어 | 풀 오케스트라, 합창 | 다큐멘터리, 역사 영상 |
| 유쾌·코믹·캐주얼 | 경쾌한 팝 / 재즈 | 관악기, 밝은 피아노 | 유머 콘텐츠, 예능 |
| 집중·효율·테크 | Lo-fi / 미니멀 일렉트로닉 | 잔잔한 비트, 피아노 루프 | 튜토리얼, 교육 콘텐츠 |
| 로맨스·설렘·따뜻함 | 재즈 발라드 / 어쿠스틱 팝 | 피아노, 통기타, 스트링 | 로맨스 브이로그, 웨딩 |
| 자연·탐험·모험 | 월드뮤직 / 앰비언트 오케스트라 | 민속악기, 자연음과 혼합 | 트래블 다큐, 야외 탐험 |

### SFX(효과음) 레이어링

| 장면 유형 | 필수 SFX | 선택 SFX | 주의 |
|-----------|---------|---------|------|
| 도시·일상 | 도로 소음, 사람 발소리 | 카페 소음, 새소리 | 과도한 SFX는 산만함 |
| 자연·야외 | 바람, 풀벌레 소리 | 시냇물, 새소리, 낙엽 밟는 소리 | 자연음은 앰비언스와 통합 |
| 긴장·액션 | 심장박동, 발소리 | 금속 충돌음, 유리 깨지는 소리 | BGM과 SFX 경쟁 주의 |
| 인터뷰·토크 | SFX 최소화 | 아주 낮은 룸톤만 | SFX가 목소리 방해 금지 |
| 코믹 | 효과음 (우당탕, 팡) | 과장된 SFX | 타이밍이 핵심 |

### 나레이션 스타일 선택

| 콘텐츠 목적 | 추천 나레이션 스타일 | 톤 | 예시 |
|------------|--------------------|----|------|
| 교육·정보 전달 | 명확하고 차분한 설명체 | 중립적, 신뢰감 | 튜토리얼, 다큐 |
| 감성 브이로그 | 일기체, 독백 스타일 | 개인적, 따뜻함 | 여행 브이로그 |
| 광고·브랜드 | 임팩트 있는 짧은 문장 | 자신감, 설득력 | CF, 브랜드 영상 |
| 역사·다큐 | 권위 있는 해설체 | 진지, 신뢰 | 역사 다큐멘터리 |
| 어린이·교육 | 친근하고 밝은 어조 | 활기차고 명확함 | 키즈 콘텐츠 |
| 나레이션 없음 | 영상과 사운드만으로 전달 | - | 감성 영상, 뮤직비디오 |

### 앰비언스 선택

| 공간/환경 | 추천 앰비언스 | 볼륨 레벨 | 역할 |
|-----------|-------------|-----------|------|
| 실내 인터뷰 | 룸톤 (낮은 화이트노이즈) | 매우 낮음 (-40dB 이하) | 공간감, 자연스러움 |
| 카페·식당 | 카페 소음 (저음 웅웅거림) | 낮음 | 분위기 설정 |
| 야외 자연 | 바람, 새소리, 물소리 | 중간 | 공간 묘사 |
| 도시·거리 | 교통 소음, 사람 소리 | 중간-낮음 | 도시 분위기 |
| 빈 공간·긴장 | 침묵 또는 극도로 낮은 드론 | 거의 없음 | 긴장감, 공허함 |

## 음악 저작권 및 라이선스 주의사항

| 라이선스 유형 | 사용 가능 여부 | 주의사항 |
|-------------|--------------|---------|
| **저작권 있는 음악** | 원칙적으로 불가 | YouTube/SNS 업로드 시 수익 박탈, 영상 차단 위험 |
| **CC0 (퍼블릭 도메인)** | 자유롭게 사용 가능 | 출처 표기 권장 |
| **CC BY** | 사용 가능 | 크레딧 표기 필수 |
| **CC BY-NC** | 비상업적만 가능 | 수익화 영상에 사용 불가 |
| **로열티 프리 (구매)** | 구매 후 사용 가능 | 라이선스 범위 확인 필수 (Epidemic Sound, Artlist 등) |
| **YouTube Audio Library** | YouTube 업로드 한정 자유 | 플랫폼 외 사용 시 확인 필요 |

**권장 음원 플랫폼**: Epidemic Sound, Artlist, Pixabay Music, Free Music Archive (FMA), YouTube Audio Library

## 기법 요약

**감성 여행 브이로그**: 어쿠스틱 BGM + 환경음 앰비언스 (바람, 발소리) + 개인적 나레이션 (선택) + SFX 최소화
**긴장 추격 장면**: 오케스트라 스트링 BGM + 심장박동 SFX + 발소리 + 앰비언스 제거
**교육 콘텐츠**: Lo-fi BGM (낮은 볼륨) + 명확한 나레이션 + 룸톤 앰비언스 + SFX 없음
**웨딩/감동**: 피아노 발라드 + 환경음 살짝 + 나레이션 없음 또는 결혼 서약 그대로

## 용어 설명

- **BGM (Background Music)**: 배경 음악. 영상 전반에 깔리는 음악
- **SFX (Sound Effects)**: 효과음. 장면에 맞는 소리를 별도로 추가
- **앰비언스 (Ambience)**: 공간 특유의 배경 소음. 룸톤, 환경음
- **룸톤**: 조용한 실내에서 녹음되는 미세한 배경 소음
- **드론 사운드**: 특정 음이 길게 지속되는 사운드. 긴장감 조성에 사용
- **로열티 프리**: 한 번 구매하면 반복 사용료 없는 음원
- **CC (Creative Commons)**: 저작자가 일부 권리를 사전에 허용하는 라이선스 체계
- **오디오 믹싱**: 여러 사운드 레이어의 볼륨과 균형을 조정하는 작업
