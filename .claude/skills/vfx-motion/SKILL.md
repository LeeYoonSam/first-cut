## 목적

영상에 VFX나 모션그래픽이 필요한 상황을 파악하고, 예산과 목적에 맞는 기법을 결정한다.

## 사용 시점

- 영상에 시각 효과(VFX) 또는 움직이는 그래픽(모션그래픽)이 필요할 때
- 예산에 맞는 VFX 수준을 결정해야 할 때
- 타이틀, 자막, 인포그래픽 등 그래픽 요소를 추가할 때
- 합성, 매트 페인팅, 크로마키 작업이 필요할 때

## 기법 요약

**VFX 도입 판단 기준:**
1. 실사 촬영으로 불가능한 장면인가?
2. 정보를 시각적으로 설명해야 하는가?
3. 브랜드 아이덴티티 강화가 필요한가?
4. 예산 내에서 충분한 품질이 나오는가?

**핵심 원칙:**
- VFX는 스토리를 돕는 도구, 목적이 아님
- 저예산 VFX 실패보다 없는 것이 낫다
- 스톡 모션 그래픽은 커스터마이징 필수
- 3D 제작은 반드시 렌더링 시간 계획에 포함

**소프트웨어 추천 경로:**
- 입문: DaVinci Resolve (무료) → Premiere Pro
- 모션 입문: After Effects 기초 → Envato Elements 스톡 활용
- 3D 입문: Blender (무료) → Cinema 4D

## 용어 설명

- **VFX (Visual Effects)**: 촬영 후 컴퓨터로 추가하는 시각 효과. 폭발, 합성, 배경 교체 등
- **모션그래픽**: 그래픽 디자인에 움직임을 더한 것. 로고 애니메이션, 인포그래픽 등
- **크로마키**: 초록색(그린스크린) 또는 파란색 배경을 제거하고 다른 배경과 합성하는 기법
- **합성 (Compositing)**: 여러 영상 레이어를 하나의 화면으로 합치는 작업
- **파티클 (Particle)**: 먼지·불꽃·연기처럼 수많은 작은 입자로 이루어진 시각 효과
- **스피드 램프**: 영상 속도를 갑자기 바꾸는 효과. 슬로우모션에서 정속, 또는 반대로
- **리깅 (Rigging)**: 3D 캐릭터에 뼈대를 만들어 움직일 수 있게 하는 작업
- **렌더링**: 컴퓨터가 3D/VFX 작업을 최종 영상으로 계산·출력하는 과정
- **키네틱 타이포그래피**: 텍스트가 음악이나 나레이션에 맞춰 움직이는 모션 기법
- **매트 페인팅**: 실사 배경을 디지털로 확장하거나 대체하는 기법
- **After Effects**: Adobe의 VFX·모션그래픽 전문 소프트웨어
- **C4D (Cinema 4D)**: 3D 모션그래픽에 많이 쓰이는 3D 소프트웨어
