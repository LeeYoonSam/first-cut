## 목적

`plans/drafts/`에 분산 저장된 씬 파일들과 중간 저장 파일들을 하나의 완성된 기획서로 통합합니다.

## 사용 시점

- 사용자가 `/export-plan`을 호출했을 때
- 메인 워크플로우(`first-cut.md`)의 7단계 최종 기획서 생성 단계에서
- 모든 씬 구성이 완료되고 최종 문서가 필요할 때

---

## 기획서 생성 지침

### 1단계: 초안 파일 수집

`plans/drafts/` 디렉토리에서 현재 프로젝트 관련 파일을 모두 읽으세요:

- `{프로젝트명}-step1.md` (사전 정보)
- `{프로젝트명}-benchmark.md` (벤치마크 분석)
- `{프로젝트명}-step2.md` (전체 구조)
- `{프로젝트명}-scene1.md`, `scene2.md`, ... (씬별 상세)
- `{프로젝트명}-최종초안.md` (있다면)

### 2단계: 최종 기획서 작성

아래 9개 섹션 구조로 기획서를 작성하세요.

## 용어 설명

- **LUT(Look Up Table)**: 색보정 프리셋. 필름 카메라 느낌, 빈티지 톤 등을 원클릭으로 적용하는 색상 설정 파일
- **타임코드**: 영상에서 특정 지점을 가리키는 시간 표시 (예: 1:30 = 1분 30초 지점)
- **앰비언스(Ambiance)**: 공간의 분위기를 만드는 배경 환경음 (카페 소음, 빗소리, 새소리 등)
