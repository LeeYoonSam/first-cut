## 목적

영상 편집에서 컷의 종류, 트랜지션 유형, 편집 리듬(페이싱)을 장면의 목적과 감정에 맞게 선택한다. 시간 경과, 감정 전환, 장소 이동 등 각 편집 목적별로 최적의 기법을 제시한다.

## 사용 시점

- 장면과 장면 사이의 전환 방식을 결정할 때
- 영상의 전체 리듬과 페이싱을 설계할 때
- 시간 경과, 장소 이동, 감정 전환을 표현할 때
- 편집 템포를 음악 비트에 맞출 때
- 특정 감정이나 긴장감을 편집으로 강화할 때

## 결정 매트릭스

### 트랜지션 유형 선택

| 전환 목적 | 추천 트랜지션 | 효과 | 사용 예시 |
|-----------|-------------|------|---------|
| 일반 장면 전환 (기본) | 스트레이트 컷 | 매끄럽고 자연스러운 연결 | 대부분의 편집에서 기본값 |
| 시간 경과 (짧음) | 점프컷 | 같은 공간에서 시간 생략 | 준비 과정, 빠른 변화 |
| 시간 경과 (긺) | 디졸브 / 페이드 | 부드러운 시간 흐름 암시 | 계절 변화, 세월 경과 |
| 꿈·회상·기억 | 디졸브 (슬로우) + 오버랩 | 현실과 비현실 경계 표현 | 플래시백, 꿈 장면 |
| 두 장면의 연결·유사성 | 매치컷 | 형태/색/움직임 일치로 연결 | 지구→오렌지, 달리는 말→달리는 차 |
| 감정적 절정·강조 | 스매시컷 | 갑작스러운 전환으로 충격 | 공포 장면, 반전 순간 |
| 장소 이동·공간 전환 | 와이프 / 슬라이드 | 공간적 이동감 표현 | 여행 영상, 장소 변경 |
| 챕터 구분·단락 | 페이드 투 블랙 | 완전한 단락 종료 신호 | 챕터 끝, 에피소드 마무리 |
| 에너지·흥분·역동 | 빠른 스트레이트 컷 연속 | 리듬감과 에너지 폭발 | 뮤직비디오, 스포츠 하이라이트 |
| 정보 제시·비교 | 스플릿 스크린 | 두 장면 동시 표시 | 비교, 동시 사건 |

### 페이싱(편집 리듬) 결정

| 페이싱 목표 | 컷 길이 | 편집 방식 | 적합한 콘텐츠 |
|-----------|--------|---------|------------|
| 긴장감 극대화 | 0.5~2초 | 빠른 컷, 점점 짧아지는 컷 | 추격, 액션, 클라이맥스 |
| 역동·에너지 | 1~3초 | 음악 비트에 맞춘 컷 | 스포츠, 뮤직비디오, 트레일러 |
| 자연스러운 드라마 | 3~8초 | 장면 내 자연스러운 흐름 | 드라마, 인터뷰, 대화 |
| 감성·여운 | 5~15초 | 롱테이크 위주, 컷 최소화 | 힐링 영상, 감동 장면 |
| 정보 전달·교육 | 3~10초 | 내용 완결 후 컷 | 튜토리얼, 설명 영상 |
| 다큐·현실감 | 가변적 | 인위적 편집 최소화 | 다큐멘터리, 리얼리티 |

### 시간 경과 표현 기법

| 시간 경과 유형 | 추천 기법 | 효과 | 예시 |
|-------------|---------|------|------|
| 수초~수분 경과 | 점프컷 | 같은 장소, 짧은 생략 | 메이크업 과정, 요리 |
| 수시간 경과 | 타임랩스 | 실제 변화를 빠르게 압축 | 일출, 도시 하루 |
| 수일~수주 경과 | 몽타주 시퀀스 | 여러 장면 빠르게 연결 | 훈련 과정, 여행 기록 |
| 수년~수십 년 경과 | 매치컷 + 디졸브 | 극적 시간 점프 | 성장 이야기, 역사 다큐 |
| 과거 회상 | 플래시백 + 색감 변화 | 현재와 과거 구분 | 기억 장면 |
| 미래 상상 | 플래시포워드 + 오버노출 | 가능성·예측 표현 | 복선, 예고 |

### 음악 비트와 편집 싱크

| 음악 특성 | 편집 방식 | 효과 |
|---------|---------|------|
| 강한 비트 (킥드럼) | 비트에 맞춰 컷 | 리듬감 극대화 |
| 멜로디 상승 | 점점 빠른 컷으로 에너지 상승 | 클라이맥스 빌드업 |
| 멜로디 하강·조용해짐 | 롱테이크로 전환 | 감정적 안정·여운 |
| 무음·침묵 | 컷 없이 정지 | 극적 긴장감 |
| 루프·반복 | 반복 구조 편집 | 일상·루틴 표현 |

## 기법 요약

**시간 경과 표현**: 타임랩스(수시간) + 몽타주(수일) + 매치컷(수년) + 디졸브(부드러운 전환)
**긴장감 구축**: 점점 짧아지는 컷 길이 + 스매시컷 (절정) + 음악 비트 싱크
**감성·힐링**: 롱테이크 + 디졸브 + 페이드 + 컷 최소화
**뮤직비디오/하이라이트**: 비트 싱크 스트레이트 컷 + 매치컷 + 빠른 편집 리듬

## 용어 설명

- **스트레이트 컷**: 아무 효과 없이 바로 연결하는 가장 기본적인 컷
- **점프컷**: 같은 앵글에서 중간을 잘라낸 컷. 시간 생략 효과
- **디졸브**: 두 장면이 겹치며 서서히 전환되는 트랜지션
- **매치컷**: 앞뒤 장면의 형태·움직임·색이 일치하는 연결
- **스매시컷**: 갑작스럽고 충격적인 장면 전환
- **페이드 투 블랙**: 화면이 천천히 어두워지며 장면 종료
- **몽타주**: 여러 짧은 장면을 빠르게 연결해 시간 경과나 아이디어를 표현
- **타임랩스**: 긴 시간을 압축해 빠르게 재생하는 기법
- **페이싱**: 편집의 속도감·리듬감. 컷 간격으로 조절
- **플래시백/플래시포워드**: 현재보다 과거/미래 장면으로 점프하는 편집
