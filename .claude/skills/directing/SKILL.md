## 목적

영상의 내러티브 구조를 선택하고, 감정 곡선을 설계하며, 시점(POV)과 미장센을 결정한다. 이야기를 어떻게 전달할지 전략적으로 연출을 계획한다.

## 사용 시점

- 영상의 이야기 구조를 기획할 때
- 감정적 흐름과 클라이맥스 위치를 설계할 때
- 시점(누구의 눈으로 볼지)을 결정할 때
- 장면의 미장센(공간·소품·색·배우 배치)을 구성할 때
- 반전, 복선, 서브플롯이 있는 이야기를 연출할 때

## 결정 매트릭스

### 내러티브 구조 선택

| 이야기 유형 | 추천 구조 | 특징 | 예시 |
|-----------|---------|------|------|
| 일반 단편·브이로그 | 3막 구조 (발단-전개-결말) | 가장 보편적, 이해 쉬움 | 여행 영상, 일상 브이로그 |
| 반전이 있는 단편 | 비선형 구조 + 제한 시점 | 정보 통제로 반전 극대화 | 미스터리, 반전 단편 |
| 감동·성장 이야기 | 영웅의 여정 (12단계) | 인물 변화와 성장 강조 | 성장 다큐, 도전 스토리 |
| 복잡한 인간관계 | 앙상블 구조 | 여러 인물의 교차 서사 | 관계 드라마, 군상극 |
| 사건 재구성·다큐 | 인 미디아스 레스 (결말 먼저) | 사건 결과부터 시작, 원인 추적 | 다큐멘터리, 실화 영상 |
| 순환·반복 이야기 | 원형 구조 | 시작과 끝이 같은 장면 | 철학적 단편, 시적 영상 |
| 짧고 강렬한 메시지 | 단일 장면 구조 | 단 하나의 장면으로 완결 | 광고, 아트 필름 |

### 감정 곡선 매트릭스

| 감정 목표 | 감정 곡선 형태 | 클라이맥스 위치 | 연출 전략 |
|---------|-------------|--------------|---------|
| 카타르시스·감동 | 점진적 상승 → 정점 → 하강 | 영상의 75~80% 지점 | 복선 누적 후 해소 |
| 반전·충격 | 평온 유지 → 급격한 반전 | 영상의 85~90% 지점 | 정보 통제, 오해 유도 |
| 긴장·스릴 | 지속적 상승 (완전 해소 없음) | 영상의 끝 또는 열린 결말 | 정보 지연, 위협 지속 |
| 힐링·위로 | 낮은 감정 시작 → 서서히 안정 | 감정 곡선 없음 (평온 유지) | 여백, 자연, 느린 페이싱 |
| 코미디·유쾌 | 기대 설정 → 반전·해소 반복 | 여러 번 반복되는 소클라이맥스 | 타이밍, 과장, 대비 |
| 교육·정보 | 호기심 유발 → 정보 제공 → 만족 | 핵심 정보 제공 직후 | 궁금증 먼저, 답 나중에 |

### 시점(POV) 선택

| 시점 유형 | 특징 | 추천 상황 | 주의사항 |
|---------|------|---------|---------|
| 전지적 시점 | 모든 인물의 내면과 상황 파악 | 복잡한 서사, 다큐 | 감정 이입 약해질 수 있음 |
| 주인공 제한 시점 | 주인공이 아는 것만 시청자도 앎 | 반전 단편, 미스터리 | 정보 통제가 핵심 |
| 1인칭 POV | 인물의 눈으로 직접 촬영 | 몰입감 극대화, FPS 스타일 | 멀미 유발 가능성 |
| 관찰자 시점 | 제3자가 관찰하는 느낌 | 다큐, 현실감 강조 | 개입 없이 사건 전개 |
| 복수 시점 교차 | 여러 인물 시점을 교대로 | 앙상블, 관계 드라마 | 시점 전환 시 명확한 신호 필요 |

### 미장센 체크리스트

| 미장센 요소 | 고려 사항 | 감정별 선택 |
|-----------|---------|-----------|
| **배경/공간** | 열린 공간 vs. 폐쇄 공간 | 자유=열린 공간, 억압=좁고 막힌 공간 |
| **색채** | 따뜻한 톤 vs. 차가운 톤 | 온기/희망=따뜻한 색, 고독/공포=차가운 색 |
| **소품** | 복선 소품, 상징 오브젝트 | 이야기와 연결된 의미 있는 소품만 배치 |
| **인물 배치** | 화면 내 인물 위치와 거리 | 갈등=인물 사이 거리 멀게, 친밀=가깝게 |
| **의상·분장** | 인물의 상태와 변화 반영 | 성장=의상 변화, 타락=점점 어두운 색 |
| **조명 방향** | 빛의 방향과 그림자 | 진실=정면 조명, 비밀=측면·역광 |
| **깊이감** | 전경/중경/배경 레이어 | 복잡한 서사=레이어 풍부, 단순=미니멀 |

### 복선(복선) 및 서브플롯 설계

| 기법 | 설명 | 사용 시점 |
|------|------|---------|
| 체이호프의 총 | 등장한 모든 소품/요소는 반드시 사용 | 중요 소품 등장 시 |
| 색채 복선 | 특정 색=특정 감정/사건 암시 | 반전 직전 색채 변화 |
| 반복 모티프 | 같은 이미지·대사·소리 반복 | 주제 강화, 반전 준비 |
| 거울·그림자 | 이중성, 내면의 다른 자아 암시 | 심리 드라마 |
| 불완전 정보 | 의도적으로 일부 정보 숨김 | 미스터리, 반전 단편 |

## 기법 요약

**반전 있는 단편**: 비선형 구조 + 제한 시점 + 정보 통제 + 복선 소품 + 클라이맥스 85~90%
**감동 성장 이야기**: 영웅의 여정 + 전지적 시점 + 점진적 감정 상승 + 따뜻한 색채 + 클라이맥스 75~80%
**힐링 브이로그**: 3막 구조 (느슨하게) + 관찰자 시점 + 평온한 감정 곡선 + 열린 공간 + 자연광
**공포·스릴러**: 제한 시점 + 지속 상승 긴장 곡선 + 폐쇄 공간 + 차가운 색채 + 체이호프의 총

## 용어 설명

- **3막 구조**: 발단(설정)-전개(갈등)-결말(해소)로 나누는 가장 기본적인 서사 구조
- **인 미디아스 레스**: 사건 중간 또는 결말부터 시작하는 서사 기법
- **영웅의 여정**: 조지프 캠벨이 정리한 12단계 서사 구조. 출발-시련-귀환
- **미장센**: 화면 안에 배치되는 모든 시각 요소의 총체 (공간, 소품, 배우, 조명, 색채)
- **POV (Point of View)**: 누구의 시선으로 이야기를 전달하는지
- **복선**: 나중에 중요해질 정보를 미리 암시해두는 기법
- **체이호프의 총**: "1막에 총이 등장하면 3막에 반드시 발사되어야 한다"는 원칙
- **앙상블**: 주인공 없이 여러 인물이 동등하게 등장하는 구조
- **카타르시스**: 감정의 정화. 관객이 영상을 통해 감정을 해소하는 경험
