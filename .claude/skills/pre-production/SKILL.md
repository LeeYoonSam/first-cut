## 목적

영상 제작 시작 전 기획 단계에서 필요한 요소를 체계적으로 정리하고 방향을 설정한다.

## 사용 시점

- 새 프로젝트 기획을 시작할 때
- 클라이언트 브리핑 후 제작 방향을 정해야 할 때
- 무드보드나 스토리보드 작성 전 구조를 잡을 때
- 촬영 전 체크리스트가 필요할 때
- 예산·일정·리소스를 계획할 때

### 타겟 분석 매트릭스

| 타겟 요소 | 질문 | 영향을 미치는 제작 요소 |
|----------|------|-------------------|
| 연령대 | 10대/20대/30대/40대 이상? | 음악 선택, 편집 속도, 언어 톤 |
| 성별 | 특정 성별 타겟인가? | 색감, 모델, 스타일링 |
| 플랫폼 | 어디서 볼 것인가? | 포맷(세로/가로), 길이, 자막 |
| 관심사 | 무엇을 좋아하는가? | 음악 장르, 비주얼 스타일 |
| 구매력 | 프리미엄/대중 타겟? | 촬영 장비, 로케이션, 배우 |
| 문화적 배경 | 국내/해외/글로벌? | 언어, 색상 상징, 문화코드 |

### 콘셉트 개발 단계

| 단계 | 내용 | 산출물 |
|------|------|--------|
| 1. 브리프 분석 | 목적·예산·일정·KPI 확인 | 브리프 요약 문서 |
| 2. 레퍼런스 수집 | 유사 사례 3-5개 분석 | 레퍼런스 링크/파일 |
| 3. 핵심 메시지 | 한 문장으로 핵심 정리 | 콘셉트 문장 |
| 4. 비주얼 방향 | 무드보드 작성 | 무드보드 (Figma/Pinterest) |
| 5. 내러티브 구조 | 시작·전개·결말 설계 | 트리트먼트 |
| 6. 스토리보드 | 장면별 시각화 | 스토리보드 (그림/사진) |
| 7. 제작 계획 | 인력·장비·일정 배분 | 제작 스케줄 |

## 기법 요약

**사전기획 핵심 원칙:**
- "왜 만드는가"를 명확히 → 모든 결정의 기준
- 레퍼런스는 반드시 클라이언트와 공유 후 방향 합의
- 스토리보드는 완성도보다 소통 도구로 활용
- 예산의 20%는 예비비로 확보
- 촬영 당일 변수를 대비해 플랜B 항상 준비

## 용어 설명

- **브리프 (Brief)**: 클라이언트가 제시하는 제작 요청서. 목적·예산·일정·타겟 포함
- **트리트먼트 (Treatment)**: 영상의 전체 내용·분위기·스타일을 글로 서술한 기획서
- **무드보드 (Moodboard)**: 영상의 색감·분위기·스타일을 이미지로 모아놓은 참고 자료
- **스토리보드**: 장면을 그림으로 나열해 영상 흐름을 시각화한 콘티 형식
- **콘티**: 스토리보드의 한국식 표현. 장면·앵글·대사·지시사항 포함
- **로케이션 헌팅**: 촬영 장소를 사전에 답사하고 적합성을 평가하는 과정
- **KPI**: 핵심 성과 지표. 영상의 성공을 측정하는 기준 (조회수, 전환율 등)
- **런다운 (Rundown)**: 행사·방송의 진행 순서표
