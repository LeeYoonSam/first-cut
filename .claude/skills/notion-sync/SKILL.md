## 목적

완성된 영상 기획서를 노션(Notion)에 페이지로 생성하여 팀원과 공유하거나 노션에서 관리할 수 있도록 합니다.

## 사용 시점

- 사용자가 `/notion-sync`를 호출했을 때
- 최종 기획서가 `plans/`에 저장된 후 노션 동기화를 원할 때
- 팀원과 기획서를 공유해야 할 때

---

## 노션 동기화 지침

### 1단계: 노션 MCP 사용 가능 여부 확인

**중요**: 노션 연동 도구는 하드코딩하지 말고, 실제 사용 가능한 도구를 먼저 확인하세요.

현재 대화에서 사용 가능한 노션 관련 MCP 도구가 있는지 확인하세요.
노션 관련 도구는 `notion`이라는 단어가 포함된 도구 이름을 가집니다.

---

### 2단계-A: 노션 MCP가 사용 가능한 경우

사용 가능한 노션 도구를 활용하여 다음 순서로 진행하세요:

#### 기획서 파일 읽기

동기화할 기획서 파일을 읽으세요:
- 사용자가 특정 파일을 지정하면 해당 파일
- 지정이 없으면 `plans/` 디렉토리에서 가장 최근 파일

#### 노션 페이지 생성

기획서의 제목을 노션 페이지 제목으로 사용하세요.

**섹션별 노션 블록 매핑**:

| 기획서 섹션 | 노션 블록 유형 |
|-------------|---------------|
| 제목 (# 헤딩) | 페이지 제목 |
| 섹션 제목 (## 헤딩) | Heading 2 |
| 씬 제목 (### 헤딩) | Heading 3 |
| 표 | Table 블록 |
| 체크리스트 (`- [ ]`) | To-do 블록 |
| 일반 텍스트 | Paragraph 블록 |
| 코드 블록 | Code 블록 |
| 굵은 글씨 (`**텍스트**`) | Bold 텍스트 |

#### 업로드 완료 후 안내

> "기획서가 노션 페이지로 생성되었습니다. 페이지 링크: [링크]
> 팀원에게 공유하려면 노션 페이지 우측 상단의 '공유' 버튼을 클릭하세요."

---

### 2단계-B: 노션 MCP가 사용 불가능한 경우

노션 MCP 도구가 없으면 다음과 같이 안내하세요:

> "현재 노션 MCP 도구가 연결되어 있지 않습니다.
>
> **노션 MCP 설정 방법**:
> 1. Claude Code 설정에서 MCP 서버를 추가하세요
> 2. Notion MCP 서버를 설치하고 API 토큰을 설정하세요
> 3. 설정 완료 후 다시 `/notion-sync`를 호출하세요
>
> **지금 당장 노션에 올리려면**:
> 1. `plans/` 디렉토리에 저장된 마크다운 기획서를 여세요
> 2. 내용을 복사하세요
> 3. 노션에서 새 페이지를 만들고 붙여넣기 하세요
>   (노션은 마크다운을 자동으로 인식합니다)
>
> 기획서 위치: `plans/{파일명}.md`"

---

## 보안 주의사항

- 노션에 업로드되는 기획서에 개인정보(이름, 연락처, 주소)가 포함되어 있는지 확인하세요
- 민감한 프로젝트 내용이 외부 서버(노션)에 저장된다는 점을 사용자에게 알려주세요
- API 키나 비밀번호는 절대 기획서에 포함하지 마세요

---

## 용어 설명

- **MCP(Model Context Protocol)**: Claude가 외부 서비스(노션, 깃허브 등)와 연동할 수 있게 해주는 연결 방식
- **노션 블록**: 노션에서 텍스트, 표, 이미지 등 각 요소를 구성하는 단위
- **API 토큰**: 외부 서비스에 접근할 때 사용하는 인증 키. 비밀번호처럼 관리해야 함
